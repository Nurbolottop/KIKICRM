class PropertyType(models.TextChoices):
    APARTMENT = "APARTMENT", "Квартира"
    HOUSE = "HOUSE", "Дом"
    LAND = "LAND", "Земельный участок"
    BUSINESS = "BUSINESS", "Бизнес помещение"
    OFFICE = "OFFICE", "Офис"
    OTHER = "OTHER", "Другое"

property_type = models.CharField(
    max_length=20,
    choices=PropertyType.choices,
    null=True,
    blank=True,
    verbose_name="Тип помещения"
)

## 🎯 Как использовать

### Создание заказа:
1. Откройте форму создания заказа
2. Заполните адрес выполнения
3. Выберите тип помещения из выпадающего списка
4. Продолжите заполнение остальных полей

### Фильтрация:
1. Откройте список заказов
2. В фильтрах найдите "Тип помещения"
3. Выберите нужный тип
4. Нажмите "Применить фильтры"

### Просмотр:
- В карточке заказа тип помещения отображается под адресом выполнения
- В таблице заказов — в отдельной колонке
