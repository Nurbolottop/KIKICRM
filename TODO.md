---

### 4. Финансовые отчеты

#### 4.1 Отчет для Основателя (FOUNDER)
**Полный доступ ко всем финансовым данным:**
- Общая выручка за период
- Расходы по категориям
- Прибыль (выручка - расходы - зарплаты)
- Начисления клинерам (выплачено/не выплачено)
- Задолженности
- Средний чек
- Конверсия по источникам
- ROI по каналам привлечения
- Экспорт в Excel/PDF

#### 4.2 Отчет для Менеджера (MANAGER)
**Операционные метрики:**
- Заказы за период (количество, сумма)
- Заказы по статусам
- Производительность клинеров
- Расходы (только просмотр)
- Средний чек
- Время выполнения заказов vs дедлайн

#### 4.3 Отчет для Оператора (OPERATOR)
**Метрики по работе с клиентами:**
- Созданные заказы за период
- Клиенты по источникам
- Конверсия заявок в заказы
- Средний чек
- Повторные заказы

#### 4.4 Отчет для SMM
**Маркетинговые метрики:**
- Клиенты по источникам (детально)
- Конверсия по каналам
- Стоимость привлечения клиента (если есть данные о расходах на рекламу)
- Повторные заказы по источникам

#### 4.5 Ежемесячный отчет (для FOUNDER)
**Автоматический отчет на 1-е число месяца:**
- Выручка за месяц
- Расходы за месяц (по категориям)
- Зарплаты клинерам (начислено/выплачено)
- Прибыль
- Количество заказов
- Средний чек
- Топ-5 клиентов по выручке
- Топ-5 клинеров по производительности
- Остатки на складе (низкий остаток)
- Экспорт в PDF + отправка в Telegram

### 5. Панель выплаты зарплаты

**Описание:** Интерфейс для выплаты накопленных начислений клинерам

**Функционал:**
- Список клинеров с накопленными суммами
- Фильтр: все / только с начислениями / только невыплаченные
- Детализация начислений по заказам
- Массовая выплата (выбрать нескольких клинеров)
- Индивидуальная выплата
- Подтверждение выплаты с указанием даты
- История выплат
- Экспорт расчетного листа в PDF

**Роли с доступом:** FOUNDER, MANAGER (только просмотр)

**Поля при выплате:**
- Дата выплаты
- Способ выплаты (наличные, карта, перевод)
- Комментарий

**После выплаты:**
- Установить `is_paid = True`
- Установить `paid_at = дата выплаты`
- Отправить уведомление клинеру в Telegram (если настроено)
